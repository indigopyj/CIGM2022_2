1. Report

TOY PROBLEM (20 PTS)

First, I load the toy image to reconstruct and then I checked its size. According to the hint, “im2var” is a function that maps each pixel to a variable number. Also, I used ‘scipy.sparse.lil_matrix’ to generate a sparse matrix for saving memery and processing time, as A.

Now I should solve the equation (Av-b)^2, where A, v, b are a sparse matrix, reconstruction image, and gradient vector respectively. Then I calculate the difference between the x-gradients of v and the x-gradients of source image. I also calculate the difference between the y-gradients of v and y-gradients of source image. Finally, I compute the difference between the colors of the top left corners of the two images.

To get v, I used the function ‘lsqr’ from scipy.sparse.linalg, which is same as ‘\’ in matlab. I reshaped the vector to visualize it as an image.

Finally, I successfully reconstruct the image.

POISSON BLENDING (50 PTS)

The main goal of poisson blending is to insert this penguin on top of the snow in the hiking image. To do this, I cropped the penguin image and generate its polygon mask by interactive jupyter notebook. I referred some open source code on the web. Here, I saved the masks as ‘.npy’ files first, then I loaded them.

Then, I cropped the penguin image and the mask for having a mimimum space.( I also referred the open source code.)

I randomly picked the location of the penguin as (777, 1991).

If I only just paste the image, the image I can get is as follows:

Then, I blend them by poisson blending.

I followed the same process as the toy problem.

(Av - b) ^ 2

A : sparse matrix

v: output image

b: desired gradient matrix

First of all, pixels outside the mask is simply same as the target pixels.

So A’s values are just 1 and b’s values are the same as the target pixel values.

However, for the pixels inside the mask should be process differently.

I utilized laplacian matrix to calculate the gradients according to each center pixel.

Laplacian matrix can be set as above.

So I set the values of A as 4 on the diagonal and -1 on the neighboring positions of the center pixel (y,x). The values of b is also set as the same way but with the source image’s values.

Note that I processed all the out-of-range error that can be occurred with if-else condition.

After this process, I applied the equation solver to get the target image.
